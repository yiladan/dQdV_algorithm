# dQdV_algorithm

#### 介绍
根据电流电压数据分析电池SOH
可用于电池组全生命周期监测


#### 使用说明

1.  流程见：final_tutorial.ipynb
2.  使用数据为恒流充电数据
3.  data数据保存在百度网盘feather格式：链接：https://pan.baidu.com/s/1fLqy2kyB0enCuSgNgbffhA 提取码：9527
4.  增加数据可视化pyecharts，需要安装pip install pyecharts==0.1.9.4
